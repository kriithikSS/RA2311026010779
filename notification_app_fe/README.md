# Campus Notification App — Stage 2

React/Next.js frontend for the campus notification platform. Built with Material UI.

## Pages

**All Notifications** — paginated list with type filters (Placement / Result / Event). Unread notifications have a blue left border and dot indicator. Click to mark as read.

**Priority Inbox** — top-N notifications ranked by type weight and recency. Configurable N via slider.
